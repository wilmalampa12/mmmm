elementos de linea (inline)y de bloque(block)

inline caracteristicas:

  • no crea nuevas lineas para cada elemento
  • no se pude definir el ancho ni el alto
  • tiene un ancho que se ajusta deacuerdo al contenido

bloque caracteristicas:

  • crea una linea para cada elemento
  • ocupan todo el espacio horizontal disponible
  • tiene un ancho y un alto
